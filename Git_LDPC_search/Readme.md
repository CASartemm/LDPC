# Проверка в Verilog:
1) В папку `c:/temp/` копировать файлы из `LDPC_Search\Файлы для tb`
данная папка содержит данные data+ldpc для каждого параметра slot size и FEC, расчетное значение сравнивается с ождаемым результатом и выводит результат в консоль .
выставление параметров режима работы происходит в тестбентче , стр.306,307
`!Отсчет FEC идет от 0 до 3 , слот сайз как есть`

Наименование самих файлов в тестбентче , например Slot_size_2_Slot_type_1.2.txt соответствует параметрам в таблице FEC TDMA.
